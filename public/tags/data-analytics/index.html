A Framework for Perception Analysis of Social Media Data During Disease Outbreaks: Uncovering Patterns of Resentment Towards Bats.

This study proposes a domain-specific machine learning framework for analyzing public perception using tweets about bats during disease outbreaks and revealing a pattern of initially prevalent anti-bat sentiments that decline over time.
